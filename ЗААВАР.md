## 1. Анх удаа суулгах

Зөвхөн **анх удаа** хийнэ.

### a) VS Code-оор livora хавтсыг нээх
**File → Open Folder → `C:\Users\Chinbat\Claude Code\livora`**

### b) Terminal нээх
**Terminal → New Terminal** (доор гарна) → доорх 2 командыг тус тусад нь ажиллуулна:

```bash
npm install
npm run dev
```

- `npm install` — багцууд татна (1–2 минут, нэг л удаа)
- `npm run dev` — сайт асаана. Терминал хаахгүй

Дараа нь броузер нээж очно:
- 🌐 Сайт: **http://localhost:3000**
- 🔐 Админ: **http://localhost:3000/admin**

Зогсооход: терминал дээр **Ctrl + C**.

## 5. 🚀 Сайтаа интернэтэд байршуулах (Vercel — үнэгүй)

### Алхам 1 — GitHub бүртгэл үүсгэх
1. https://github.com → **Sign up** → имэйлээрээ бүртгүүлнэ.
2. Имэйлээ баталгаажуулна.

### Алхам 2 — GitHub дээр шинэ repo үүсгэх
1. https://github.com/new руу орно.
2. **Repository name:** `livora`
3. **Public** сонгоно (үнэгүй).
4. **README, .gitignore** битгий тэмдэглэ (бид аль хэдийн үүсгэсэн).
5. **Create repository** дарна.
6. Дараагийн хуудсанд гарсан **HTTPS URL**-ыг хуулж аваарай — `https://github.com/таны-нэр/livora.git` гэж байх ёстой.

### Алхам 3 — Кодоо GitHub руу оруулах
VS Code → Terminal:

```bash
cd "C:\Users\Chinbat\Claude Code\livora"
git init
git add .
git commit -m "Анхны хувилбар"
git branch -M main
git remote add origin https://github.com/таны-нэр/livora.git
git push -u origin main
```

⚠️ Хэрэв `git` тушаал танигдаагүй бол: https://git-scm.com/download/win → татаж суулгана.

⚠️ GitHub нэр, нууц үг асуухад **Personal Access Token** (нууц үг биш) бичих ёстой:
- https://github.com/settings/tokens → **Generate new token (classic)** → бүх `repo` checkbox сонго → үүсгэсэн токеныг хуулж нууц үгийн оронд буулга.

### Алхам 4 — Vercel-д холбох
1. https://vercel.com → **Sign Up** → **Continue with GitHub**.
2. Vercel-д GitHub-аа холбоно.
3. **Add New → Project** → жагсаалтаас `livora` repo-г **Import**.
4. **Configure Project** дэлгэц гарна:
   - **Framework:** Next.js (автоматаар таних ёстой)
   - **Root Directory:** өөрчлөхгүй
   - **Environment Variables** хэсгийг **дарж нээгээд** доорх 4-ийг нэг нэгээр нь нэмнэ:

| Name | Value |
|---|---|
| `ADMIN_USER` | `Chinbat` |
| `ADMIN_PASSWORD` | `Chinbat*0521.` |
| `GMAIL_USER` | `livora.mn@gmail.com` |
| `GMAIL_APP_PASSWORD` | `iyhl ivrz mype yzto` (зайтайгаа эсвэл зайгүй) |
| `ADMIN_EMAIL` | `livora.mn@gmail.com` |

5. **Deploy** дарна. 1–2 минут хүлээнэ.
6. Дуустал `https://livora.vercel.app` гэх мэт хаягтай **жинхэнэ интернэт сайт** гарна! 🎉

### Алхам 5 — Өөрийн домэйн залгах (сонголтоор)

## 7. 🗄️ SQL өгөгдлийн сан холбох (Supabase)

Та "SQL дээр base үүсгэмээр байна" гэсэн. Дараах алхмаар үүсгэнэ:

### Алхам 1 — Supabase бүртгэл
1. https://supabase.com → **Start your project** → GitHub-ээр нэвтэрнэ.
2. **New project** → нэр `livora` → нууц үг сонго → **Asia Southeast (Singapore)** регион.
3. 2 минут хүлээнэ.

### Алхам 2 — Хүснэгт үүсгэх
**SQL Editor → New query** дотор доорхийг буулгаад **RUN** дарна:

```sql
-- Курсууд
create table courses (
  id uuid primary key default gen_random_uuid(),
  slug text unique not null,
  title text not null,
  short_description text,
  description text,
  category text not null,
  price integer not null default 0,
  cover_image text,
  duration_minutes integer,
  lessons integer,
  level text,
  instructor text,
  rating numeric,
  students_count integer default 0,
  is_published boolean default true,
  created_at timestamptz default now()
);

-- Курсын файлууд
create table course_files (
  id uuid primary key default gen_random_uuid(),
  course_id uuid references courses(id) on delete cascade,
  name text not null,
  type text not null,
  size_mb numeric,
  url text not null
);

-- Захиалгууд
create table orders (
  id uuid primary key default gen_random_uuid(),
  course_id uuid references courses(id),
  customer_name text not null,
  customer_email text not null,
  customer_phone text not null,
  amount integer not null,
  bank_used text,
  status text default 'pending',
  created_at timestamptz default now()
);

-- Нийтлэлүүд
create table articles (
  id uuid primary key default gen_random_uuid(),
  slug text unique not null,
  title text not null,
  excerpt text,
  content text,
  cover_image text,
  category text,
  read_minutes integer,
  author text,
  published_at timestamptz default now()
);
```

### Алхам 3 — API key авах
**Project Settings → API** → доорхи 2 утгыг хуулж аваарай:
- `Project URL`
- `anon public` key

### Алхам 4 — Кодод холбох
Энэ хэсэгт хүрэх үед **надад мэдэгдээрэй**, би Supabase-тай холбож, mock data-г жинхэнэ өгөгдлөөр сольж өгье. (`@supabase/supabase-js` суулгаж, data.ts-ийг async fetch болгоно.)
